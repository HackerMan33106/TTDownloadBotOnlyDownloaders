# 🎬 TTDownloadBot - Only Downloaders

**Telegram бот для скачивания контента из TikTok и других социальных сетей с поддержкой файлов до 2GB.**

## ✨ Возможности

### 📹 Скачивание контента
- 🎥 **Видео** - TikTok, YouTube, Instagram, Facebook и другие платформы
- 📸 **Слайдшоу** - TikTok фото-карусели с музыкой
- 🎵 **Аудио** - извлечение звука из любого видео
- 🔗 **Множественные ссылки** - отправьте несколько URL в одном сообщении
- 🧹 **Умная обработка** - автоматическая очистка и развёртывание коротких ссылок

### 🌐 Поддерживаемые платформы
TikTok • YouTube • Instagram • Facebook • Twitter/X • Reddit • SoundCloud • VK • Rutube • Dzen • Bilibili • PornHub

### 🎛️ Удобное управление
- 🗑️ **Удаление контента** - кнопка на каждом сообщении
- 🎵 **Скачать аудио** - извлечение звука одним нажатием
- 🔄 **Быстрая отправка** - кэширование для мгновенной повторной отправки
- 📊 **Статистика** - команда /storage для проверки использования диска

### 🔐 Система доступа
- 👥 **Белый список** - контроль доступа для пользователей и групп
- 📊 **Лимиты** - дневные ограничения на использование
- 🚫 **Чёрный список** - блокировка нежелательных пользователей
- 👮 **Админы** - полный доступ к управлению ботом

### 💾 Кэширование
- ⚡ **Мгновенная отправка** - повторные запросы обрабатываются за секунды
- 🎵 **Аудио кэш** - сохранение извлечённых аудио
- 📸 **Слайдшоу кэш** - быстрая отправка фото-каруселей

## 🔧 Команды бота

- `/start` - Приветствие
- `/ping` - Проверка задержки
- `/help` - Справка по командам
- `/dw <url>` - Скачать видео с любой платформы
- `/wl` - Управление белым списком (админ)
- `/bl` - Управление лимитами (админ)
- `/admin` - Управление админами (админ)
